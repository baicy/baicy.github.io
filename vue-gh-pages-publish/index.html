Vue项目发布在github预览

发表于 2019-09-05

前端项目想发布在网上供大众预览有很多种方法，比如将要使用的利用gh-pages发布项目

使用的Vue.js版本是3.11.0，版本不同会有些微的文件结构不同

以自己的项目arkObserver为例，预览地址

1. 编译项目

在项目根目录新建文件vue.config.js，将publicPath更改为

默认路径是空，gh-pages会自动使用

2.上传github

有2种情况

1.项目代码同时发布在github上，利用branch gh-pages预览

2.不发布项目代码，只部署branch master预览

本例属于情况1，已有github respository

3.gh-pages配置

在github项目仓库选择Settings-Options-GitHub Pages

Source处选择gh-pages branch，稍等一会，会显示Your site is published at http://xxx

部署完成

# Vue # github # 发布
不熟练总会出点事，记安装配置php+redis
